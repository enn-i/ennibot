# Ennibot: A Discord Bot made using (Concord)[https://github.com/Cogmasters/concord/tree/master?tab=readme-ov-file]


### Instructions to run:

- install (Concord)[https://github.com/Cogmasters/concord/tree/master?tab=readme-ov-file]
- run `make` in the directory to setup dependencies
- open `config.json` and replace the "token" field with your bot token
- clone the repository into the directory
- run `make` to start the bot.
